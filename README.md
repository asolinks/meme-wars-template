# 🎮 Git Meme Wars - Team Repository

Welcome to your team's meme headquarters!

## Quick Start

1. Clone this repository
2. Add your memes to the `memes/` folder
3. Update `team-info.json` with your team details
4. Commit and push your changes
5. Watch them appear on the live dashboard!

## Rules

- Maximum 5 memes per team
- Keep content appropriate
- Use proper Git workflow (branches, commits, PRs)
- Have fun!
